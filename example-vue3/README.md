# example-vue3

이 샘플 코드는 vue cli 기반으로 vue3 프로젝트를 생성 후 RealReport 미리보기를 구현한 예제입니다.
[Vue cli 공식문서](https://cli.vuejs.org/)

## vue 프로젝트 생성
```bash
npm install -g @vue/cli
vue create example-vue3
```

### realreport 패키지 설치
```bash
npm install realreport
```

### Vue 예제 로컬환경 실행

```bash
npm i
npm run serve
```
